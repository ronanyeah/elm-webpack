# elm-webpack

## Requirements
- [npm CLI](https://docs.npmjs.com/)

## How to install
- `npm install --force`
> Flag required due to [dependency conflict](https://github.com/klazuka/elm-hot-webpack-loader/issues/21).

## How to build
- `npm run build`

## How to develop
- `npm run watch`
- Navigate to [localhost:8000](http:localhost:8000).
